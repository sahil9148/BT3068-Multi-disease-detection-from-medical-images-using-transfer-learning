The Problem

Why This Project Matters

The global shortage of radiologists and the exponential growth of medical imaging data has created an urgent diagnostic crisis.

🏥

Radiologist Shortage

Rural hospitals in developing nations often have zero resident radiologists, leaving critical diagnoses delayed or missed entirely.

🔬

Single-Disease Pipelines

Existing AI tools are narrow — a model for chest X-rays can't assess brain tumours or diabetic retinopathy simultaneously.

🎯

Our Solution

A unified multi-disease detection system covering 4 domains with explainable AI, achieving 95.67% accuracy without massive compute.

Technical Architecture

How It Works

A two-phase transfer learning pipeline with channel attention and ensemble classification.

INPUT
Medical Image
380×380 px
→
BACKBONE
EfficientNetB4
ImageNet pre-trained
→
ATTENTION
SE Block
Channel re-calibration
→
HEAD
Dense + Softmax
Disease classification
→
XAI
Grad-CAM
Explainability overlay
⚡

Two-Phase Training

Phase 1: Freeze backbone, train SE block + head at lr=1e-4.
Phase 2: Unfreeze top 100 layers, fine-tune at lr=1e-5.

🔗

Ensemble Framework

MobileNetV2 + DenseNet121 + ResNet101 combined via Stacking Meta-Learner and MRLA optimization for 3× performance gain.

🔍

Explainability (XAI)

Grad-CAM heatmaps and LIME explanations show exactly which image regions triggered each diagnosis.
